# OSI Model (Open System Interconnection Model)

# Layers
OSI Model have 7 layers:
(Plese Do Not Tell Secret Password Anyone)
- **Application Layer (L7)**
- Presantation Layer (L6)
- Session Layer (L5)
- **Transport Layer (L4)**
- Network Layer (L3)
- **Data Link Layer (L2)**
- Physical Layer (L1)

# But in Real life SWE use these 3 layers:
- Application Layer (L7)
- Presantation Layer (L6)
- Session Layer (L5)
